Esempio 3!

Esempio con l'utilizzo di jms-write ed jms-foreach per creare, ciclare porzioni di html

In questo esempio creamo dinamicamente un Nav menu, e un table popolando l'header e le colonne

Nav menu!

<li jms-foreach="menu.links" class="nav-item active">
<a class="nav-link" for-property-href="links.url" for-property="links.title"> </a>
</li>

Header table!

<th scope="col" jms-foreach="table_header" for-property="table_header.titolo"></th>

Table Rows e column!

<tr jms-foreach="table_data">
<td for-property="table_data.regione"></td>
<td for-property="table_data.provincia"></td>
<td for-property="table_data.comune"></td>
<td for-property="table_data.dugt@@ @@topografico"></td>
<td for-property="table_data.cap"></td>
</tr>
