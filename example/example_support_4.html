Esempio 2

Esempio con l'utilizzo di jms-foreach per ciclare e creare porzioni di html

In questo esempio creamo dinamicamente tag html <li> ed <a> per creare il Nav menu.

<li jms-foreach="menu.links" class="nav-item active">
<a class="nav-link" for-property-href="links.url" for-property="links.title"> </a>
</li>
jms-foreach="menu.links" per ciclare e creare un tag html <li> per ogni elemento dell'array dedicata alla creazione del menu l'utilizzo di for-property-href="links.url" per settare la property href del tag <a> e for-property="links.title" per settare l'html del tag <a>
